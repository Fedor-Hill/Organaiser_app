# "Organaiser" project

It`s improve interaction to see your marks and other data from Kundelik.kz.

## Installation
```bash
git clone https://github.com/Fedor-Hill/Organaiser_app.git 
cd Organaiser_app 
flutter gen-l10n
flutter run 
```
